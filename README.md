# AI Pet Health

基于 AI 的宠物健康管理平台，帮助宠物主人更好地了解和管理宠物健康。

## 技术栈

- **Frontend**: React + TypeScript + Vite
- **Backend**: Python + FastAPI
- **AI**: LLM-based health analysis

## 项目结构

```
ai-pet-health/
├── frontend/          # React 前端应用
├── backend/           # Python FastAPI 后端
├── docs/              # 项目文档
└── README.md
```

## 开发环境

### 前端
```bash
cd frontend
npm install
npm run dev
```

### 后端
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
